   Alias for :option:`-copy-origin` :option:`-copy-orientation` :option:`-copy-size`.

.. option:: -reset

   Set orientation, origin, and affine transformation matrix to default.

.. option:: -reset-dof

   Set affine transformation matrix to default.

.. option:: -dofin <file>

   Apply transformation to axis, spacing and origin information
   in the header. Note that any shearing that is present is
   stored as additional affine transformation (c.f. -putdof).

.. option:: -putdof <file>

   Store affine transformation in image header (NIfTI only).

.. option:: -swapxy

   Swap the x and y axis vectors.

.. option:: -swapxz

   Swap the x and z axis vectors.

.. option:: -swapyz

   Swap the y and z axis vectors.
