Arguments
---------

.. option:: target

   Target point set.

.. option:: source

   Source point set.


Command options
---------------

.. option:: -dofin <file>

   Target point set transformation. (default: none)

.. option:: -closest-point

   Find the closest point in the source data set. (default)

.. option:: -closest-cell

   Find the closest cell point in the source data set.

.. option:: -cor <name>

   Name of correspondence type, e.g., 'closest point', 'closest cell'.

.. option:: -corpar <name> <value>

   Set parameter of chosen correspondence type.

.. option:: -feature <name> [<weight>]

   Use named point data as feature. (default: spatial coordinates)

.. option:: -corin <file>

   Read correspondences from text file which lists on each line
   the indices of corresponding target and source points.

.. option:: -corout <file>

   Write indices of corresponding points to text file.

.. option:: -output <file>

   Write point set of corresponding points.

.. option:: -vertices

   Add a vertex cell for each point.

.. option:: -vectors

   Add displacement vectors from corresponding output points to
   input target points. (default: off)

.. option:: -indices

   Store indices of corresponding points as point data.
   Indices are cell indices in case of :option:`-closest-cell`.

.. option:: -copy-pointdata

   Copy input point data to output point set. (default: off)

.. option:: -copy-celldata

   Copy input cell data to output point set. (default: off)

.. option:: -compress, -nocompress

   Whether to compress XML VTK file data. (default: on)

.. option:: -binary

   Write legacy VTK file in binary format. (default)

.. option:: -ascii

   Write legacy VTK file in ASCII  format.
