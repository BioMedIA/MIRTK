::

    calculate-boundary-map <domain> <codomain> <output> [options]


Description
-----------

.. include:: _descriptions/calculate-boundary-map.rst



Arguments
---------

.. option:: domain

   Tesselation of the surface of the input shape.

.. option:: codomain

   File name of piecewise linear complex or name of primitive shape:
   
   - disk (default)
   - square
   - polygon

.. option:: output

   Computed boundary map.


Command options
---------------

.. option:: -radius <float>

   Radius of primitive codomain shape. When non-positive, the radius
   is chosen such that the area matches the area of the input surface.
   (default: 0)

.. option:: -uniform

   Uniformly distribute boundary points.

.. option:: -chord-length

   Map boundary curve proportionally to the boundary edge length.

.. option:: -subdivided

   Map each sub-segment defined by the selected points to the same
   output curve length with chord-length parameterization of each
   selected boundary sub-segment. (default)

.. option:: -select <id>...

   Indices of selected surface points. (default: none)
