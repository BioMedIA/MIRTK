   Combine current selection criterium to the left of this option
   with the following criteria using a logical AND operator. (default)

.. option:: -or

   Combine current selection criterium to the left of this option
   with the following criteria using a logical OR operator.

.. option:: -eq <value>

   Select cells whose data value is equal the specified value.

.. option:: -ne <value>

   Select cells whose data value is not equal the specified value.

.. option:: -lt <value>

   Select cells whose data value is less than the specified value.

.. option:: -le <value>

   Select cells whose data value is less than or equal the specified value.

.. option:: -gt <value>

   Select cells whose data value is greater than the specified value.

.. option:: -ge <value>

   Select cells whose data value is greater than or equal the specified value.

.. option:: -surface, -nosurface

   Output surface of resulting point set. (default: off)

.. option:: -normals, -nonormals

   Recalculate normals of output surface. (default: off)

.. option:: -origids <name>

   Add original cell IDs as output cell data array of given name. (default: off)
