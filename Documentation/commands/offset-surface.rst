Command options
---------------

.. option:: -offset <distance>

   Offset distance. (default: 0)

.. option:: -relative

   Multiply given distance value by length of bounding box diagonal.

.. option:: -implicit

   Use offset surface reconstructed from implicit surface model.

.. option:: -size <nx> [<ny> <nz>]

   Number of voxels to use for implicit surface model.
   (default: bounding box divided by :option:`-voxelsize`)

.. option:: -voxelsize <dx> [<dy> <dz>]

   Size of voxels to use for implicit surface model.
   This option is ignored when image :option:`-size` is specified.
   (default: length of bounding box diagonal divided by 512)

.. option:: -surface <file>

   Reference surface to which a minimum offset distance
   to the input surface is required. (default: <input>)

.. option:: -scalars <name> <min> [<max>]

   Only move points whose given scalars point data
   is within the given range.
