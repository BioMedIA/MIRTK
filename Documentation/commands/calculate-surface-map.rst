calculate-surface-map
=====================

.. program:: calculate-surface-map


Synopsis
--------

::

    calculate-surface-map <input> <output> [options]


Description
-----------

.. include:: _descriptions/calculate-surface-map.rst



Arguments
---------

.. option:: input

   Point set delineating the boundary of the map domain or name
   of primitive shape such as: "disk", "square", or "sphere".

.. option:: output

   File path of output map. A piecewise linear map is stored as VTK file.
   Other maps are stored in a custom binary format.


Output options
--------------

.. option:: -barycentric

   Use spring constants based on generalized barycentric coordiantes.

.. option:: -mean-value

   Use spring constants based on mean value coordinates.

.. option:: -conformal

   Conformal surface map or as-conformal-as-possible volumetric map.

.. option:: -harmonic

   Harmonic volumetric map.


Command options
---------------

.. option:: -p <n>

   Exponent of harmonic energy term. When non-positive, solve for an
   approximate harmonic surface map using a spring network. (default: 0)

.. option:: -name <string>

   Name of point data array used as fixed point map.  (default: tcoords)

.. option:: -mask <string>

   Name of point data array used as fixed point mask. (default: boundary)

.. option:: -max-iterations <n>

   Maximum no. of linear solver iterations. (default: 1 or size of problem)
