Output options
--------------

.. option:: -rigid

   Output rigid transformation.

.. option:: -similarity

   Output similarity transformation.

.. option:: -affine

   Output affine transformation.

.. option:: -affine-mffd

   Output affine transformation as global component of MFFD.

.. option:: -mffd

   Output multi-level free-form deformation (MFFD).

.. option:: -ffd

   Output free-form deformation (FFD).

.. option:: -svffd

   Output diffeomorphic free-form deformation (SV FFD).

.. option:: -translations, -notranslations

   Allow/disallow translations.

.. option:: -rotations, -norotations

   Allow/disallow rotations.

.. option:: -scaling, -noscaling

   Allow/disallow scaling.

.. option:: -shearing, -noshearing

   Allow/disallow shearing.


Command options
---------------

.. option:: -dofin <dof>

   Read affine parameters from input transformation.

.. option:: -orientation <x1> <x2> <x3>  <y1> <y2> <y3>  <z1> <z2> <z3>

   Set upper 3x3 (rotation) matrix entries
   (e.g., image orientation from image info output).

.. option:: -image-reset <file>

   Set transformation to difference between spatial image to world mapping
   of given image and the default lattice with axes parallel to the axes of
   the world coordinate system and image center at the origin. The affine
   output transformation maps points of the unoriented world system with
   origin at the image center to the world coordinates of the input image.

.. option:: -image-rotation <file>

   Set transformation to difference between spatial image to world mapping
   of given image and an image lattice with axes parallel to the axes of
   the world coordinate system. The affine output transformation maps points
   of the unoriented world system to the world coordinates of the input image.
   Hence, the domain of the map is the same as the world domain of the image,
   except that the orientation matrix is equal the identity matrix.

   Translation along x axis in mm.             (default: 0)

.. option:: -ty <ty>

   Translation along y axis in mm.             (default: 0)

.. option:: -tz <tz>

   Translation along z axis in mm.             (default: 0)

.. option:: -rx <rx>

   Rotation around x axis in degrees.          (default: 0)

.. option:: -ry <ry>

   Rotation around x axis in degrees.          (default: 0)

.. option:: -rz <rz>

   Rotation around x axis in degrees.          (default: 0)

.. option:: -sx <sx>

.. option:: -sxy <sxy>

.. option:: -target <image>

   Target image. Required when output is a FFD. (default: none)

.. option:: -source <image>

.. option:: -Tp <value>

.. option:: -align-top

   Align top-most slices of :option:`-target` and :option:`-source` image. (default: center)

.. option:: -displacements, -disp <pset1> <pset2> [<cor12>]

   Create transformation which minimizes the mean squared distance between pairs
   of fiducial point sets <pset1> and <pset2>. When a single <pset> is given,
   it creates a transformation which approximates the displacement vectors stored
   in the "vectors" point set attributes array. Option can be used multiple times.

.. option:: -approximate <dofin>

   Create transformation which approximates
   another given transformation.
   Option can be used multiple times.
