smooth-surface
==============

.. program:: smooth-surface


Synopsis
--------

::

    smooth-surface <input> <output> [options]
    smooth-surface <input> <output> <iterations> [lambda] [mu] [options] (deprecated)


Description
-----------

.. include:: _descriptions/smooth-surface.rst



Arguments
---------

.. option:: input

   File name of input surface mesh.

.. option:: output

   File name of output surface mesh.

.. option:: iterations

   No. of smoothing iterations. (deprecated: see :option:`-iterations` option)

.. option:: lambda

   Relaxation/scale factor for odd  iterations. (deprecated: see -lambda option)

.. option:: mu

   Magnitude of scale factor for even iterations. (deprecated: see :option:`-mu` option)


Command options
---------------

.. option:: -iterations <n>

   Number of smoothing iterations. (default: 1)

.. option:: -lambda <float>

   Relaxation/scale factor for odd  iterations. (default: 1)

.. option:: -mu <float>

   Relaxation/scale factor for even iterations. (default: lambda)

.. option:: -points

   Smooth node positions. (default if no :option:`-scalars` specified)

.. option:: -scalars [<name>]

   Name of scalar point data array to smooth. If no <name> given,
   all non-attribute scalar arrays and the SCALARS if any are smoothed.

.. option:: -exclnode

   Exclude node itself from smoothing, only average adjacent values.

.. option:: -inclnode

   Include node itself in the smoothing.

.. option:: -areaweighted

   Area weighted Laplacian relaxation of surface node positions. (default)

.. option:: -windowedsinc [<band>]

   Adjust point positions using a windowed sinc interpolation function.

.. option:: -combinatorial

   Combinatorial weighting of adjacent nodes.

.. option:: -distance [<sigma>]

   Inverse distance weighted smoothing kernel.

.. option:: -gaussian [<sigma>]

   Gaussian smoothing kernel.

.. option:: -anisotropic ([<sigma>] [<tensor_array>] | <sigma1> <sigma2> [(<e2_array> | <e1_array> <e2_array>)])

   Anisotropic Gaussian smoothing kernel given input tensor field point data array
   named <tensor_array>. The default tensor field used with no arguments or only <sigma> specified
   is the curvature tensor output of the calculate-surface-attributes -tensor command. Alternatively,
   specify <sigma1> and <sigma2> together with the direction of maximum curvature in <e2_array> or both,
   the direction of minimum and maximum curvature direction in point data arrays named <e1_array> and <e2_array>.

.. option:: -track [<name>]

   Track the signed distances traversed by points (positive is outwards).
   The optional arguments specifies the name of output output point data array.
   (default: "smoothingDists" if option is given without <name>).

.. option:: -threshold <value>

   A value indicating the smoothness as the L_2 norm of H^2 over the surface.
   See Tosun, MedIA, 2004. Iterations stop if the norm for the surface drops
   below the given value. Only used for :option:`-areaweighted` smoothing.
