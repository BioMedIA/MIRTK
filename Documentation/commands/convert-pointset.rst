convert-pointset
================

.. program:: convert-pointset


Synopsis
--------

::

    convert-pointset <input>... <output> [options]


Description
-----------

.. include:: _descriptions/convert-pointset.rst



Arguments
---------

.. option:: input

   Input  point set file (.vtk, .vtp, .vtu, .stl, .ply, .off, .dfs, .obj).

.. option:: output

   Output point set file (.vtk, .vtp, .vtu, .stl, .ply, .off, .dfs, .node, .poly, .smesh).


Command options
---------------

.. option:: -merge

   Merge points of non-polygonal input point sets. (default: off)

.. option:: -holes

   Add a point inside the input surface meshes, except off the first input mesh,
   to the hole list of the output .poly or .smesh file. (default: no holes)

.. option:: -nocelldata

   Do not write cell  data to output file. (default: off)

.. option:: -nopointdata

   Do not write point data to output file. (default: off)

.. option:: -ascii

   Write legacy VTK files encoded in ASCII. (default: off)

.. option:: -binary

   Write legacy VTK files in binary form. (default: on)

.. option:: -compress

   Compress XML VTK files. (default: on)

.. option:: -nocompress

   Do not compress XML VTK files. (default: off)
