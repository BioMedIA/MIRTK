Command options
---------------

.. option:: -h, --help

   show this help message and exit

.. option:: -a MEANS [MEANS ...], --ages MEANS [MEANS ...], --means MEANS [MEANS ...]

   Discrete time points for which to construct atlas.

.. option:: -s SIGMA [SIGMA ...], --sigma SIGMA [SIGMA ...]

   Standard deviation of temporal kernel(s).

.. option:: -o OUTDIR, --output OUTDIR, --outdir OUTDIR

   Output directory of final atlas.

.. option:: -w TMPDIR, --workdir TMPDIR, --tmpdir TMPDIR

   Working directory for intermediate files.

.. option:: -e ENERGY, --energy ENERGY

   Energy function of image registration (e.g., 'asym',
   sym', 'ic').

.. option:: -c CHANNELS [CHANNELS ...], --channels CHANNELS [CHANNELS ...]

   Images to use for multi-channel registration.

.. option:: -m MEASURES [MEASURES ...], --measures MEASURES [MEASURES ...]

   Image (dis-)similarity measure to use for each
   channel.

.. option:: -b BENDING [BENDING ...], --bending BENDING [BENDING ...]

   Bending energy weight for each step.

.. option:: --bins BINS

   No. of bins to use for histogram-based measures.

.. option:: --window WINDOW

   Local window size for NCC measure in voxels.

.. option:: --spacing SPACING [SPACING ...]

   Control point spacing for each step.

.. option:: -j JACOBIAN [JACOBIAN ...], --jacobian JACOBIAN [JACOBIAN ...]

   Jacobian penalty weight for each step.

.. option:: --bch-terms BCHTERMS  No. of BCH terms to use for composition of SV FFDs.


.. option:: -g GROWTH, --growth GROWTH

   Whether to joinedly estimate mean shape and change.

.. option:: -i START, --start START

   Continue construction after the specified step.

.. option:: -n STEPS, --steps STEPS

   Number of steps of iterative atlas
   construction/refinement. Use 'iterations' config entry
   if not specified (default: 10).

.. option:: -q LONGQUEUE, --queue LONGQUEUE, --long-queue LONGQUEUE

   Name of batch system queue. Use 'condor' for HTCondor.
   Otherwise, the argument is assumed to be the name of a
   SLURM partition.

.. option:: --short-queue SHORTQUEUE

   Name of batch system queue to use for short running
   jobs (about 1-30 min). Use --long-queue by default.

.. option:: -t THREADS, --threads THREADS

   Maximum number of CPU cores/threads to use.

.. option:: -v VERBOSE, --verbose VERBOSE

   Verbosity level of output messages: 0) no output, 1)
   report progress, 2) print command arguments.
