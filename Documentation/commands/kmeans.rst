Description

.. include:: _descriptions/kmeans.rst



Input options
-------------

.. option:: -mask <mask>

   run k-means inside the provided mask

.. option:: -replicates <number>

   number of random initializations, the best is retained (default: 10).

.. option:: -iterations <number>

   maximum number of iterations per replicate  (default: 100).

.. option:: -saveprobs <basename>

   output probabilities. These are computed by initialising a Gaussian distribution for each cluster with the mean and variance of the points.

.. option:: -blurprobs <sigma>

   The output probabilities are blurred with a kernel of size sigma.
