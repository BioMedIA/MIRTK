Arguments

Command options
---------------

.. option:: -input-labels, -input <file>

   Input segmentation label image, use either this option or <input> argument.
   For each of the following options which take a <labels> argument,
   the respective segmentation mask is derived from this input label
   image by merging the specified segments of this brain segmentation.
   Multiple integer labels can be specified as the <labels> argument,
   each separated by at least one space character. A range of labels
   can be specified as <first>..<last>. When the <labels> argument is
   a single argument that is neither an integer nor a label range,
   the argument is expected to be the name of a binary mask image file.
   This option is required when any of the other options has one or more
   <labels> arguments. It is ignored otherwise. (default: none)

.. option:: -output-labels, -output <file>

   Output label image, use either this option or <output> argument.

.. option:: -hemispheres <file>

   Hemispheres mask (0: outside, 1: right, 2: left) from
   which cutting plane is computed. (default: none)

.. option:: -right-hemisphere, -rh <file>|<labels>

   Mask or labels of structures of right hemisphere used to compute cutting plane.
   This option is required when no :option:`-hemispheres` mask is given.

.. option:: -left-hemisphere, -lh <file>|<labels>

   Mask or labels of structures of left  hemisphere used to compute cutting plane.
   This option is required when no :option:`-hemispheres` mask is given.

.. option:: -subcortical, -sb <file>|<labels>

   Subcortical / deep brain structures segmentation mask or labels.
   When specified, these structures are cut into RH and LH. (default: none)

.. option:: -white-matter, -wm <file>|<labels>

   White matter segmentation mask or labels. (default: none)

.. option:: -grey-matter, -gm <file>|<labels>

   Grey matter segmentation mask or labels. (default: none)

.. option:: -brainstem, -bs <file>|<labels>

   Brainstem segmentation mask or labels. (default: none)

.. option:: -cerebellum, -cb <file>|<labels>

   Cerebellum segmentation mask or labels. (default: none)

.. option:: -closing, -closing-iterations <n>

   No. of iterations used to close holes between right/left subcortical,
   brainstem, and cerebellum segmentations. (default: 0)

.. option:: -subcortical-closing <n>

   No. of iterations used to close holes between right/left subcortical segmentation. (default: 0)

.. option:: -brainstem-closing <n>

   No. of iterations used to close holes in brainstem segmentation. (default: 0)

.. option:: -cerebellum-closing <n>

   No. of iterations used to close holes in cerebellum segmentation. (default: 0)

.. option:: -brainstem-and-cerebellum, -cerebellum-and-brainstem, -bscb, -cbbs [on|off]

   Whether to merge brainstem and cerebellum. (default: off)
