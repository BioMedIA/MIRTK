Description

.. include:: _descriptions/smooth-image.rst



Arguments
---------

.. option:: input

   Input image. Multiple channels/frames are smoothed independently.

.. option:: output

   Blurred output image.

.. option:: sigma

   Standard deviation of Gaussian smoothing kernel. When negative, its
   absolute value is multiplied by the average voxel size. (default: 1)


Command options
---------------

.. option:: -3D

   Blur image in all spatial directions. (default)

.. option:: -4D

   Blur image in all dimensions.

.. option:: -x

   Blur image in x dimension.

.. option:: -y

   Blur image in y dimension.

.. option:: -z

   Blur image in z dimension.

.. option:: -short

   Set data type of output to short integers.

.. option:: -float
